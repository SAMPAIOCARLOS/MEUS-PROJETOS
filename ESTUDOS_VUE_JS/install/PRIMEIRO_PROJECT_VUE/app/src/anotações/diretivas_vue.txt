AS DIRETIVAS EM VUE_JS SÃO ATRIBUTOS ESPECIAIS QUE É COLOCADO EM ELEMENTS DO HTML 
PARA ATRIBUIR ALGUM COMPORTAMENTO DINAMICO DE ACORDO COM ALGUMA VERIFICAÇÃO POR EXEMPLO

OS MAIS COMUNS SÃO
1) v-show
2) v-if
3) v-else-if
4) v-else

1)Alterna a visibilidade de um elemento com base em uma expressão, modificando 
o estilo display (N CASO DE ACORDO COM O RESULTADO DE ALGUMA CONDIÇÃO O ELEMENTO 
SOME OU APARECE)

2) Renderiza um bloco condicionalmente. Se a expressão for verdadeira, o bloco será renderizado (NO CASO
TAMBÉM O ELEMENTO PODE SUMIR OU APARECER DEPENDENDO DO RESULTADO DE UMA CONDIÇÃO)

A DIFERENÇA ENTRE O "v-show" E O "v-if" É QUE O "v-show" SE O RESULTADO FOR FALSO ELE VAI APLICAR NO ELEMENTO 
A PROPRIEDADE "display: none" VAI FAZER  ELEMENTO SUMIR, MAS ELE VAI CONTINUAR EXISTINDO,
JÁ O "v-if" FAZ  ELEMENTO DESAPARECER COMPLETAMENTE DO CODIGO

3) O "v-for" SERVE PARA ITERAR SOBRE ARRAYS OU ARRAYS DE OBJETOS PARA PODER CRIAR ELEMENTOS DINAMICAMENTE
É UM FOR NORMAL DO JAVASCRIPT
